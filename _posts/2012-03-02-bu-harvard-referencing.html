This file is intended to be used in Microsoft Word for the referencing function built into Word. It is intended to be used with the Bournemouth University method of Citing References and has been painstakingly edited to be as accurate as possible to the referencing detailed in that file.

  • To install on Windows, navigate to your winword.exe directory. Put the file in the BibliographyStyle folder.
  • To install on Mac, find Microsoft Word in your applications and select Show Package Contents. Put the file in the Contents/Resources/Style/ folder.

I cannot offer any more assistance in using the file and creating a bibliography, although this Microsoft help file should assist you enough. I recommend before saving your document out for final marking that you convert your bibliography to static text by right clicking on it.

While it is as accurate as possible, this referencing system may not be perfect. There may be errors, oversights, or incorrect formatting of references. If you notice any issues, please feel free to contact me. I cannot be held responsible for any errors in your bibliography. You should still double check your bibliography. This software is intended to make your life easier, not lazier. It is intended for you to be able to make your references better, make the marker's life easier, and allow you to concentrate on writing an awesome dissertation/essay.

If you are unsure what to do with the Referencing tool in Word, it may be best to do your referencing manually.

If you find this tool has saved you a lot of time, feel free to drop me a line on Twitter or email. Just knowing you've benefitted is a great boost. And if you're feeling especially generous, maybe you can buy me pizza. Or a cider.
